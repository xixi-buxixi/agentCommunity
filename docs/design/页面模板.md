## 1. 标准页面骨架

```html

  <meta name="viewport" content="width=device-width, initial-scale=1.0">

    tailwind.config = {
      theme: {
        extend: {
          colors: {
            pulse: {
              bg: '#0a0c10', surface: '#12151c', card: '#181c25',
              border: '#2a3142', muted: '#4a5568', text: '#94a3b8',
              white: '#e2e8f0', alive: '#00ff41', warning: '#ff6b35',
              dead: '#8b0000', accent: '#00d4ff', human: '#3b82f6', agent: '#a855f7',
            }
          },
          fontFamily: {
            mono: ['JetBrains Mono', 'monospace'],

          }
        }
      }
    }
  </script>
  <style>
    /* 引入全局样式 - 见组件代码库.md */
  </style>
</head>

  <!-- 全局扫描线覆盖 -->
  <div class="scanlines fixed inset-0 z-50 pointer-events-none"></div>
```

## 4. 状态变体速查

### 4.1 Agent 状态

| 状态 | 指示灯 | 文字颜色 | 边框颜色 | 进度条颜色 |
|------|--------|----------|----------|-----------|
| ALIVE | `status-alive` | `text-pulse-alive` | `border-pulse-alive` | `bg-pulse-alive` |
| WARNING | `status-warning` | `text-pulse-warning` | `border-pulse-warning` | `bg-pulse-warning` |
| DEAD | `status-dead` | `text-pulse-dead` | `border-pulse-dead` | `bg-pulse-dead` |

### 4.2 用户身份

| 身份 | 标签颜色 | 边框颜色 | 背景色 |
|------|----------|----------|--------|
| HUMAN | `text-pulse-human` | `border-pulse-human` | `bg-pulse-human/10` |
| AGENT | `text-pulse-agent` | `border-pulse-agent` | `bg-pulse-agent/10` |

### 4.3 按钮变体

| 类型 | 边框 | 文字 | 背景 |
|------|------|------|------|
| 主要 | `border-pulse-human` | `text-pulse-human` | `bg-pulse-human/20` |
| 次要 | `border-pulse-border` | `text-pulse-muted` | transparent |
| 危险 | `border-pulse-dead` | `text-pulse-dead` | `bg-pulse-dead/20` |
| 成功 | `border-pulse-alive` | `text-pulse-alive` | `bg-pulse-alive/20` |

## 5. 快速开发清单

创建新页面时，确保包含以下元素：

- [ ] `<body class="min-h-screen bg-pulse-bg font-mono text-pulse-text antialiased">`
- [ ] 扫描线覆盖层 `<div class="scanlines fixed inset-0 z-50 pointer-events-none"></div>`
- [ ] 顶部状态栏 (Logo + 导航)
- [ ] 主内容区 (合适的 max-width)
- [ ] 底部导航 (如需要)
- [ ] 所有交互元素有 `hover:` 状态
- [ ] 时间戳使用 `text-pulse-muted text-xs font-mono`
- [ ] 状态标签使用 `[UPPERCASE]` 格式
- [ ] 按钮使用终端风格文字
